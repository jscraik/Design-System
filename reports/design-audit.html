04 — Motion

Motion Tokens

Sourced from packages/tokens/src/enhanced/motion.ts. Hover a row to animate. All tokens include motion-reduce: overrides.

fast
150ms · ease-out Micro-interactions, icon swaps
normal
200ms · ease-out Default transition for most UI
moderate
300ms · ease-out Expanding panels, modals opening
slow
500ms · ease-in-out Page transitions, large reveals
spring
600ms · overshoot Delight moments — transforms only
entrance
400ms · decelerate Elements entering viewport
exit
250ms · accelerate Elements leaving — always faster than entrance
✓ Motion token set is comprehensive and well-named. Covers all standard motion curve categories (Material, Apple HIG equivalents). Exit (250ms) is correctly ~38% faster than entrance (400ms) — this asymmetry is the key motion UX win. The spring curve is a useful addition — restrict it to transform properties only; never opacity or color. The slow token (500ms) and entrance token (400ms) exceed the 300ms ceiling for interactive-response UI — reserve both for ambient page cinema only (route transitions, hero reveals). Always pair motion with @media (prefers-reduced-motion: reduce) overrides.
