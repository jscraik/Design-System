## Architecture Summary

### Core Packages

- **packages/ui**: React component library (main output)
- **packages/tokens**: Design tokens (CSS variables, Tailwind preset)
- **packages/runtime**: Host adapters (embedded/standalone)
- **packages/json-render**: JSON rendering utilities
- **packages/effects**: Animation/effects library
- **packages/widgets**: Standalone widget bundles for ChatGPT
- **packages/cli**: CLI tools
- **packages/astudio-icons**: Icon library

### Platforms

- **platforms/web**: Web apps (reference app + Storybook)
- **platforms/mcp**: MCP server for ChatGPT integration
- **platforms/desktop**: Desktop apps (Tauri)

### Key Dependencies

- `@design-studio/tokens` → tailwind configs, UI library
- `vite`, `vitest`, `@vitejs/plugin-react` → build/test tooling
- `@playwright/test` → e2e/a11y testing
- `@openai/apps-sdk-ui` → visual foundation

### Risk Areas (from harness.contract.json)

- **High**: `packages/widgets/src/widgets/auth`, `packages/runtime`, `packages/cloudflare-template`, `platforms/mcp`
- **Medium**: `packages/ui`, `packages/tokens`, `packages/json-render`, `packages/effects`, `packages/cli`
- **Low**: tests, docs, examples, storybook
