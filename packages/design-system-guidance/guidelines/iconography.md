# Iconography Guidance

## Table of Contents
- [Rule](#rule)
- [Do](#do)
- [Avoid](#avoid)

## Rule
Use the canonical icon source defined by the UI package (`@design-studio/ui/icons`).

## Do
- Import icons from `@design-studio/ui/icons`.
- Keep icon usage consistent with accessibility requirements (labels/titles when needed).

## Avoid
- Deprecated `@design-studio/astudio-icons` imports.
- Mixing multiple icon systems in one surface unless explicitly documented.
