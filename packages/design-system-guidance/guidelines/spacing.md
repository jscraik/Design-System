# Spacing Guidance

## Table of Contents
- [Rule](#rule)
- [Do](#do)
- [Avoid](#avoid)

## Rule
Spacing must come from spacing tokens or mapped spacing utilities.

## Do
- Use spacing scale tokens (for margin, padding, gap).
- Reuse semantic layout spacing patterns.

## Avoid
- Raw `px` spacing values in UI code.
- One-off spacing values that break rhythm across components.
