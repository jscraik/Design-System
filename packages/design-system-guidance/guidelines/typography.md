# Typography Guidance

## Table of Contents
- [Rule](#rule)
- [Do](#do)
- [Avoid](#avoid)

## Rule
Typography should use tokenized font sizes, line heights, weights, and tracking values.

## Do
- Use typography utilities or CSS variables backed by tokens.
- Keep responsive type scales aligned with the design system.

## Avoid
- Raw `px` font sizes in component code.
- Ad-hoc line-height values not represented in tokens.
