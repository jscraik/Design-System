# @design-studio/astudio-icons

> **Deprecated: Use `@design-studio/ui/icons` instead**

This package is deprecated in favor of `@design-studio/ui/icons`. All icons have been migrated to the canonical icon system.

## Migration

To migrate your imports:

```bash
# Before
import { Icon } from "@design-studio/astudio-icons/react/platform";

# After
import { Icon } from "@design-studio/ui/icons";
```

## Status

**Maintenance Only**: No new features will be added to this package.

## Questions?

See migration guide at `/packages/ui/src/icons/` in your codebase or contact the Design System team.

---

**Note**: This package remains published for backward compatibility, but new projects should use `@design-studio/ui/icons`.
